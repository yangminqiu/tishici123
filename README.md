# 提示词360

一个帮助用户通过上传图片生成 AI 绘画提示词的工具。

## 功能
- 支持拖拽上传图片
- 支持预览上传的图片
- 支持 Midjourney、Stable Diffusion、DALL·E 等主流 AI 绘画工具
- 用户登录和注册功能

## 技术栈
- 前端：原生 HTML/CSS/JavaScript
- 认证：Firebase Authentication
- 部署：待定

## 本地开发
1. 克隆项目
2. 运行 `live-server` 启动开发服务器
3. 在浏览器中访问 `http://localhost:8080`

## 注意事项
- 图片大小限制为 5MB
- 支持的图片格式：JPG、PNG、GIF
